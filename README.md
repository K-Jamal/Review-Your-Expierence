# Review Your Experience

## Benodigdheden
    Xampp Control Panel

## Hoe open je het project
Open Xampp control panel

Start Apache en MySql op

Maak een nieuwe database aan met de naam ```rye1```

Importeer de sql vanuit de backup directory

Nu staat de database klaar

Verplaats de ```Review-Your-Experience``` map naar ```htdocs```

Nu kan je de map localhosten

Ga naar de browser

Zoek op localhost en dan de naam van de map

Bijv. localhost/Review-Your-Experience

Nu zie je de website
